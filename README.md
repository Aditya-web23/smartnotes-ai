# smartnotes-ai
SmartNotes AI is an AI-powered Notes Generation Platform designed to help students create structured, exam-ready notes instantly.

This project allows students to:
- Register for access (Free or Paid)
- Join community (Free plan)
- Get direct AI access (Paid plan)
- Automatically store user data in Google Sheets
